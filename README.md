c00
Conceitos:
1 - Função 'main', entrada e saida básica
	:> Toda execução em C começa obrigatóriamente pela função 'main'. Ela é o ponto de entrada do seu programa.
	:> Retorno: O mais comum é um 'int' (inteiro). O valor '0' sinaliza ao SO (sistema operacional) que o programa terminou com sucesso.
	:> Protótipo Simples:
		'int main(void)
		{
			// Seu código aqui
			return (0);
		}'

2 - Saída de Dados: 'putchar' esse comando é importante para entender a manipulação de apenas um caractere por vez na saida padrão (stdout).
	:> Uso: 'putchar(\'A\')'; imprime a letra A.
	:> Esse aprendizado é fundamental para entender como strings são formadas e cadeias de caracteres são terminadas em '\0'.

3 - Controle de Fluxo
	:> O controle de fluxo é a tomada de desisões do programa.
		Condicionais ('if/else')
	:> Serve para avaliar se uma declaração é verdadeira ou falsa.
		'if (n > 0)
			write(1, "+", 1);
		else
			write(1, "-", 1);'
